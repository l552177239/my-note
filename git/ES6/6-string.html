第六小节：字符串的扩展

字符串的扩展

字符串模版（template string）

字符串模板是增强版的字符串，用反引号（`）标识。它可以当作普通字符串使用，也可以用来定义多行字符串，或者在字符串中嵌入变量。

模板字符串中嵌入变量，需要将变量名写在${}之中。

let obj = {name:'Liu',age:'24'};
`my name is :${obj.name}`

大括号内部可以放入任意的JavaScript表达式，可以进行运算，以及引用对象属性。

var a = 3;
var b = 2;
`运算：${a} + ${b} = ${a + b}`
